### 工具函数

this._util.funcName(param);

```javas
示例：

工具函数：日志打印

用法：
let data = {
	name: '张三',
	age: 18,
	hobby: '篮球'
}
this._util.log('人物描述', data);

结果：
---------------------人物描述---------------------
{name: "张三", age: 18, hobby: "篮球"}
```

**新增、查看、删除、修改请前往`libs/util/index.js`**
